
# APIs de consulta de placa de veículos

Aqui será listado algumas APIs de consulta de placas de veículos, seja motos ou carros

## PlacaAPI

Nossa API é um serviço web .NET ASMX, que permite que você conecte-se a partir de qualquer ambiente de programação, seja .NET (C#, Visual Basic .NET), ou através de qualquer linguagem de programação que suporte o SOAP (PHP NuSOAP, Python, Ruby, Java etc.)

Registre-se gratuitamente e vamos configurá-lo com uma conta de teste, onde você pode fazer 10 pesquisas de veículo gratuitamente.

Se você precisar de maior volume de pesquisas, então a nossa taxa é de R$ 0,80 por pesquisa, compras em blocos de um mínimo de 100, e um desconto de 10% está disponível para pacotes de mais de 1000.

Diferencial: **Agora oferecemos uma API que pode acessar o histórico dos veículos, assim seus clientes poderão automaticamente ver problemas recorrentes do carro, ou além disso se resguardar através de testes aprovados ao comprar um carro. Essa funcionalidade em particular funciona apenas para o Reino Unido, e oferecemos um serviço de busca unificado na maioria dos países europeus, incluindo, Dinamarca, Finlândia, Países Baixos, Noruega, Suécia, Portugal, Itália, França e Espanha.** 

| Nome da API | Link da API |
|------------|-------------|
| PlacaAPI | [PlacaAPI](https://www.placaapi.com/) |

## API Brasil

Um hub de APIs que oferece consulta de placas de carro. Permite até sete consultas gratuitas por dia para testar a solução, com planos pagos a partir daí. Fornece dados como marca, modelo, cor, ano e situação cadastral.

Diferencial: **Oferece inúmeras opções de APIs para consultas veiculares**

| Nome da API | Link da API |
|-------------|-------------|
| API Brasil | [API Brasil](https://www.apibrasil.com.br/)

## Rex Consulta veicular

Esse serviço afirma oferecer laudo completo: histórico de roubo/furto, restrições jurídicas e financeiras, multas/débitos, leilões, sinistros, dados de cadastro (RENAVAM, chassi, motor), e também preço referência via FIPE.

| Nome da API | Link da API |
|-------------|-------------|
| Rex Consulta Veicular | [Rex Consulta Veicular](https://rexconsulta.com.br/)

## Historicar

Disponibiliza pacotes de consultas que combinam dados de registro, restrições, débitos/multas, roubo/furto, lembrança de recall, gravames, histórico FIPE, indícios de sinistro ou leilão (alguns inclusive com fotos). 

| Nome da API | Link da API |
|-------------|-------------|
| Historicar | [Historicar](https://www.historicar.com.br/)
